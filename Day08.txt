    len(집합객체) : 집합객체의 요소(element,E) 개수
    len(문자열) : 문자열의 문자 수

    str.format() friends
    (1) find(문자열) : 문자열을 찾고 어디에 위치하는지 인덱스를 반환
    (2) lstrip,rstrip,strip : 왼쪽, 오른쪽, 양쪽으로부터 문자열을 찾아서 삭제, 타언어의 ltrim, rtirm, trim
    (3) title : 각 단어의 첫 글자를 대문자로 만들어주는 것

    a="abcdefg"
    a[0] = 'a', a[4] = e
    a[시작값:종료값:증감값] = a의 시작값 인덱스부터 종료값 인덱스 전까지 증감값만큼 뛰어서
    a[0::] - n1:n2:n3가 정상이나 n2와 n3가 생략.
             n1이나 n2가 생략되면 unbounded, n3가 생략되면 1
             0부터 무제한 끝까지, 1씩 증가하여 => a 본인
             a[::] - 얘도 a 본인
             a[::1] : 0부터 무제한끝까지 1씩 증가하여
             a[::-1] : 무제한끝부터 0까지 1씩 감소하여

2. 집합객체 복사하기
    a = b[1:5] => b[1] ~ b[4] 복사
    c = d[1:] => d[1]부터 unbounded(끝)까지 복사
    e = {1,2,3,4,5}
    f = e[1:] => 오류, 집합은 순서가 없음
